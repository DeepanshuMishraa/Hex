Add hardware audio muting and app-aware AI post-processing. Muting now uses Core Audio hardware mute on all output devices (including Bluetooth) for instant, glitch-free silence during recording. New App-Aware AI mode detects the active application and formats transcriptions contextually: commands in Terminal, emails in Mail, code in editors, messages in Slack/Discord, and more. Browser URL detection enables Gmail formatting when on mail.google.com.
